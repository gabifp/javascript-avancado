XML HTTP Request (XHR)

XML HTTP Request (XHR) é um objeto JavaScript que permite o envio e a recepção de dados de um servidor web usando o protocolo HTTP ou HTTPS. O XHR é o principal componente por trás da abordagem Ajax.

Principais características do XHR:

  • Criação do objeto: Para utilizar o XHR, é necessário criar uma instância do objeto XMLHttpRequest através do construtor "new XMLHttpRequest()".
  • Enviar solicitações: Após a criação do objeto XHR, é possível usar os métodos fornecidos por ele para enviar solicitações para o servidor. O método mais comum é o "open()", que configura a solicitação com o método HTTP desejado e o URL do servidor, e o método "send()", que envia a solicitação para o servidor.
  • Manipulação de respostas: O objeto XHR permite definir funções de callback para lidar com a resposta do servidor. É possível especificar uma função para ser executada quando a resposta estiver pronta, bem como funções para lidar com erros e progresso da solicitação.
